# Joke Generator App

Small Angular project to practice creating services and making API calls using HttpClient.

## Learned

- Creating services in Angular
- Generating services using Angular CLI
- Understanding @Injectable and providedIn: 'root'
- Using provideHttpClient() to enable API calls
- Injecting dependencies through the constructor
- Working with HttpClient to fetch data from an API
- Using Observables to handle asynchronous data
- Subscribing to service methods inside components
- Handling loading and error states
