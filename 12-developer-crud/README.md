# Developer Crud App

Angular CRUD application focused on data flow between components and service-oriented data handling.

## Learned

- setInterval(): do action after n milliseconds.
- Parent to Child Communication: passing parameters from parent component to child component using @Input().
- Child to Parent Communication (No parameter): using @Output() and EventEmitter to trigger functions in the parent.
- Child to Parent Communication (With parameter): using $event to pass data from child to parent via EventEmitter.
- Creating DTOs: using Omit to create Data Transfer Objects for API requests.
